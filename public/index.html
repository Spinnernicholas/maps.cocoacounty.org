Instructions

Select desired race in the menu in the bottom.

  • Opacity: Set the precinct transparency so you can see the map
  • First Dropdown: Select Race
  • Second Dropdown:
    • WINNER BY PRECINCT: Highlights the winner in each precinct
    • Candidate/Choice: Heatmap | The darker the precinct, the higher the vote percentage
